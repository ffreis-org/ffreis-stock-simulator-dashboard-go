# Security Policy

## Reporting a Vulnerability

To report a security vulnerability in this project, please use GitHub's private advisory feature:

**[Report a vulnerability](https://github.com/FelipeFuhr/ffreis-stock-simulator-dashboard-go/security/advisories/new)**

Please do not open a public GitHub issue for security vulnerabilities. Using the private advisory link above ensures the report is handled confidentially.

Once a report is received, we will:

1. Acknowledge receipt of the vulnerability report.
2. Investigate and assess the severity of the issue.
3. Release a fix and disclose the vulnerability details after it has been resolved.
